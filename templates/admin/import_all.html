Important: Follow the Correct Import Order

  1. Users First: Always import users before importing any other data.
  2. Images Second: Import images after users are imported.
  3. Comments Last: Import comments only after both users and images are imported.

Following this order will prevent errors from missing references.

Note: Importing images without the referenced users will fail. Similarly, importing comments requires both the referenced users and images to exist in the system.
